# gemini-clean
Find and remove reduntant gemini references

## Install
```
$ npm install gemini-clean
$ gemini-clean --help
```

## Example
```
$ gemini-clean example.js
```

```
root
root/parent
  plain
  hovered
  pressed
root/parent/child
  plain
  hovered
root/parent/child/grandChild
  plain
  pressed
```
